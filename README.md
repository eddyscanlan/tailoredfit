# TailoredFit - Free ATS Resume Score Checker

> Check your resume's ATS compatibility score for free. Get an AI-optimized resume for $7 (one-time, no subscription).

## What it does

TailoredFit is a free ATS (Applicant Tracking System) resume checker. Paste your resume and a job description, and instantly see:

- **ATS Match Score** - How compatible your resume is with the job description
- **Keyword Gap Analysis** - Which keywords from the job description are missing from your resume
- **AI Optimization** ($7 one-time) - Get a rewritten resume that naturally integrates all missing keywords
